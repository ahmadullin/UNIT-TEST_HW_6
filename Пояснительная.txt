1.	Тест testCalculateAverage: Этот тест покрывает следующие сценарии:
1.1.	Создание четырех разных списков с разными размерами и значениями.
1.2.	Вызов метода calculateAverage для каждого из списков.
1.3.	Сравнение ожидаемых результатов с теми, которые возвращаются методом calculateAverage.

2.	Тест testCompareLists: Этот тест покрывает следующие сценарии:
1.1.	Создание пары из четырех разных списков.
1.2.	Вызов метода compareLists для этой пары.
1.3.	Сравнение вывода метода compareLists с ожидаемым результатом. В целом, тесты покрывают различные сценарии использования приложения и помогают убедиться в правильной работе его функциональности в этих условиях.
